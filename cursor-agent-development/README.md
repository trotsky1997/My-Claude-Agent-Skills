# Cursor Agent Development Skill

这个技能包含了使用 `cursor-agent` CLI 进行交互式开发的完整知识。

## 技能内容

- **基本命令**：交互模式、非交互模式、会话管理
- **开发-验收循环模式**：监督式开发流程
- **高级特性**：模型选择、输出格式、沙盒模式、浏览器自动化
- **MCP 集成**：配置和管理 MCP 服务器
- **最佳实践**：提示词技巧、迭代开发、会话持久化
- **常见模式**：项目初始化、功能添加、代码审查

## 使用场景

当需要：
- 使用 cursor-agent 进行项目开发
- 实现监督式开发-验收循环
- 管理多个开发会话
- 配置 MCP 服务器
- 优化 agent 交互流程
